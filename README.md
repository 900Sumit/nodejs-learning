# Node.js Learning Project

## Process

1. git clone https://github.com/strivolve/nodejs-learning.git
2. cd nodejs-learning
3. Ensure Node.js is installed (`node -v`)
4. Run example:
   - node 3_asynchnorous.js
   - node app.js
   - node 6_app.js
   - node 7_server.js
   - node 9_server_routing.js
   - node 11_form_submit.js

5. Update:
   - git add .
   - git commit -m "Update student info and add README"
   - git push
