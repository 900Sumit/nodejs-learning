# Node.js Learning Project

This repository contains simple Node.js demos for callbacks, files, servers, forms, and routing.

## Clone from Git

```bash
# clone repository
git clone https://github.com/strivolve/nodejs-learning.git
cd nodejs-learning
```

## Install prerequisites

No dependencies are required for these plain Node.js examples, but ensure you have at least Node.js installed.

Check Node:
```bash
node -v
```

## Run examples

From repository root, run with Node:

```bash
node 3_asynchnorous.js
node app.js
node 6_app.js
node 7_server.js
node 9_server_routing.js
node 11_form_submit.js
```

For the server examples, open browser at `http://localhost:3000` or configured port.

## Notes

- `3_asynchnorous.js` writes and reads files asynchronously.
- `9_server_routing.js` shows basic routing and HTML responses.
- `form.html` / `form1.html` are used by the form input examples.
